Local server instructions

Options to serve the site locally from this folder:

1) PowerShell helper (recommended on Windows)

    .\serve.ps1

2) Python (if installed)

    python -m http.server 8000
    # or
    py -m http.server 8000

3) npm (uses npx to run http-server)

    npm run start

Open http://localhost:8000 in your browser. Stop the server with Ctrl+C in the terminal.
